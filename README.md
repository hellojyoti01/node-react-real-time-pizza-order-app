## Install

```sh
# for npm
npm install

# for yarn
yarn install

# for pnpm
pnpm install
```

## Setup

Copy the .env.example to .env and update the below environment value

```sh
cp .env.example .env
```

```sh
# .env
PORT=4000
MONGO_URI=mongodb://localhost/node-react-real-time-pizza-order-app
COOKIE_SECRET=thisismysecret

```

## Usage

```sh
# for npm
npm run start

# for yarn
yarn run start

# for pnpm
pnpm run start
```
